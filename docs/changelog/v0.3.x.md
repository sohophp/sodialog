# SoDialog v0.3.8

发布日期：2026-06-24

## Added

- 新增 Modal header 与关闭按钮配置：`hideHeader`、`hideCloseButton`、`closeButtonLabel`、`closeButtonText`。
- Modal Lab 补齐 header、关闭按钮、尺寸、拖动、滚动、footer、诊断与 lifecycle/action hook 参数。

## Changed

- 顶部导航收敛为核心入口，更新日志、版本与开发笔记移动到页脚。
- 示例中心按任务拆分为独立页面，并保持文档页左侧导航一致。
- 组件内页头部更聚焦必要介绍，移除不准确或低价值的动作与状态文案。

## Fixed

- 修复 Context Menu 打开 Modal 时面板出现浏览器默认黑色焦点框的问题。
- 修复 Offcanvas 示例嵌套旧网页导致体验混乱的问题。

## Deprecated

暂无。

## Removed

暂无。

## Upgrade Notes

`hideHeader`、`hideCloseButton`、`closeButtonLabel`、`closeButtonText` 均为新增可选项，旧代码无需迁移。
