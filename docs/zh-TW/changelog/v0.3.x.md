# SoDialog v0.3.8

發布日期：2026-06-24

## Added

- 新增 Modal header 與關閉按鈕配置：`hideHeader`、`hideCloseButton`、`closeButtonLabel`、`closeButtonText`。
- Modal Lab 補齊 header、關閉按鈕、尺寸、拖動、滾動、footer、診斷與 lifecycle/action hook 參數。

## Changed

- 頂部導覽收斂為核心入口，更新日誌、版本與開發筆記移到底部。
- 範例中心按任務拆分為獨立頁面，並保持文件頁左側導覽一致。
- 元件內頁 header 更聚焦必要介紹，移除不準確或低價值的動作與狀態文案。

## Fixed

- 修復從 Context Menu 開啟 Modal 時面板出現瀏覽器預設黑色焦點框的問題。
- 修復 Offcanvas 範例嵌套舊 demo 頁造成體驗混亂的問題。

## Upgrade Notes

所有新增 Modal 選項都是可選項，既有程式碼不需要遷移。
