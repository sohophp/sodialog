# SoDialog v0.3.6

發布日期：2026-06-23

## Added

- 新增更清楚的文件資訊架構，涵蓋文件、元件、API、範例、指南、更新日誌、版本與開發筆記。
- 新增 Modal Lab，可調整設定並產生程式碼。
- 新增元件頁頭、版本徽章、API 參數表、Lab 卡片與 CDN 範例。

## Changed

- Sidebar 依內容區域拆分，開發筆記不再混入元件、API 或指南側邊欄。
- 文件與可執行範例中的 CDN 版本更新到目前版本。

## Fixed

- 恢復 Blog / 開發筆記在主導覽與頁腳中的入口。
- 修復本地化 Blog 圖片無法顯示的問題。

## Upgrade Notes

本次主要是文件站修復與體驗調整，不需要遷移核心執行時 API。
