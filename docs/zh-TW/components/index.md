# 元件

<VersionBadge />

SoDialog 元件基於瀏覽器原生 DOM 能力實作，並保持零執行時必需依賴。

<div class="sod-doc-grid-compact">
  <a class="sod-doc-tile" href="/zh-TW/components/modal"><h3>Modal</h3><p>確認、表單、Promise 流程與需要阻斷背景操作的對話框。</p></a>
  <a class="sod-doc-tile" href="/zh-TW/components/offcanvas"><h3>Offcanvas</h3><p>篩選、詳情、設定面板與行動端操作區。</p></a>
  <a class="sod-doc-tile" href="/zh-TW/components/toast"><h3>Toast</h3><p>狀態回饋、佇列訊息、重複策略與 live region 通知。</p></a>
  <a class="sod-doc-tile" href="/zh-TW/components/context-menu"><h3>Context Menu</h3><p>右鍵選單、鍵盤導覽、typeahead 與選單觸發 Dialog。</p></a>
</div>

## 其它元件 {#other-components}

目前公開套件以 Modal、Offcanvas、Toast 和 Context Menu 為核心元件。框架適配器應保持薄封裝，並放在核心執行時之外。
