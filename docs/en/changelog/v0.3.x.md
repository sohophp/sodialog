# SoDialog v0.3.8

Release date: 2026-06-24

## Added

- Added Modal header and close button options: `hideHeader`, `hideCloseButton`, `closeButtonLabel`, and `closeButtonText`.
- Expanded Modal Lab coverage for header, close button, size, dragging, scrolling, footer, diagnostics, and lifecycle/action hooks.

## Changed

- Simplified the top navigation around core sections and moved changelog, versions, and notes to the footer.
- Split the examples center into task-focused pages and kept the left sidebar consistent across docs pages.
- Tightened inner page headers by removing inaccurate quick actions and low-value version badges.

## Fixed

- Fixed the browser default black focus outline shown when opening a Modal from a Context Menu.
- Fixed the Offcanvas example being embedded through the old demo page.

## Upgrade Notes

All new Modal options are optional, so existing code does not need a migration.
