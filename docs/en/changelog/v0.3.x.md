# SoDialog v0.3.7

Release date: 2026-06-23

## Added

- Added a clearer documentation architecture for docs, components, API, examples, guides, changelog, versions, and notes.
- Added Modal Lab with live configuration and generated code.
- Added component headers, version badges, API parameter tables, Lab cards, and CDN examples.

## Changed

- Split sidebars by section so Blog notes no longer appear inside component, API, or guide sidebars.
- Updated examples and CDN snippets to the current package version.

## Fixed

- Restored Blog discoverability from the main navigation and footer.
- Fixed Blog images in localized pages.

## Upgrade Notes

No runtime API migration is required for this documentation-focused release.
