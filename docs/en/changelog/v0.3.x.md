# SoDialog v0.3.9

Release date: 2026-06-25

## Added

None.

## Changed

None.

## Fixed

- Fixed the component-page Offcanvas Playground being constrained inside an iframe preview.
- Fixed the browser default black focus outline that could appear when opening a confirmation dialog from a context menu.
- Fixed small Modal drag jitter before movement starts, and a possible position shift when pressing the drag handle again after a drag.

## Upgrade Notes

This is a bug-fix release. Existing code does not need a migration.
